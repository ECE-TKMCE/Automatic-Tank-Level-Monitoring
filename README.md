# 💧 Automatic Water Tank Level & Quality Monitoring System

## 📌 Project Description

This project focuses on automating the everyday task of water tank management by combining **water level detection**, **automatic pump control**, and **basic water quality monitoring** into a single reliable system. In many households, switching the water motor ON and OFF is still done manually, which often leads to issues like **overflow, dry running, and water wastage**. This system eliminates that problem by making the process fully automatic and efficient.

At the core of the system is a **microcontroller-based control unit** that continuously monitors the tank using **float level sensors** placed at different levels. When the water level drops below a predefined minimum level, the system automatically turns ON the pump using a relay module. Once the tank reaches the maximum level, the pump is switched OFF, ensuring safe and efficient operation without human intervention.

To extend the functionality beyond just level control, the system also integrates a **TDS (Total Dissolved Solids) sensor** to monitor water quality in real time. This allows users to get an idea of the water condition while the tank is being filled. The values can be displayed or monitored through serial output, making the system more informative and useful in practical scenarios.

## ⚙️ Key Features

* ✅ Automatic water level detection using float sensors
* ✅ Relay-based pump control (ON/OFF automation)
* ✅ Prevention of overflow and dry running
* ✅ Real-time TDS-based water quality monitoring
* ✅ Low-cost and easy-to-implement design
* ✅ Scalable for IoT and remote monitoring extensions

## 🛠️ Implementation Stages

The project was developed in multiple stages:

1. **Breadboard Prototype**
   Initial testing was done on a breadboard to validate the logic and working of sensors, relay, and microcontroller. This helped in identifying issues like unstable connections and noise.

2. **PCB Implementation**
   After successful testing, the circuit was transferred to a custom PCB for improved reliability, compactness, and long-term use.

3. **3D Enclosure Design**
   A protective enclosure was designed using Autodesk Fusion to house all components safely. The final system is placed inside a 3D-printed case for durability and practical deployment.
